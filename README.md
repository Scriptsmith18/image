# 📸 Image Popup Project

A simple and responsive image popup (modal) viewer built using HTML, CSS, and JavaScript. Click on any image to view it in a larger format with a smooth overlay effect.

## 🧰 Features

- Responsive image gallery layout
- Smooth popup animation
- Click outside the image or on close icon to dismiss
- Easy to integrate into any webpage
